# PyPublica
PyPublica is an open source tools to help people find open acces scientific article based on filter. PyPublica support more than 1000+ paper/request with batch download feature. PyPublica made using python with tkinter as the GUI.
<br>
## How its work?
PyPublica utilizes public search query APIs provided by Elsevier and NCBI. To ensure a seamless downloading experience, PyPublica also incorporates Unpaywall API calls. Consequently, users are required to have an Elsevier Developer account and API key, as well as an NCBI account and API key. To help you get started, the creator has provided a tutorial below.
